jCustom Placeholder v.0.1

Author: Tony Bispo Pinheiro aka Tony Pine

Description

The jCustom Placeholder emulate the placeholder effect in older browsers
like IE7. In the other Modern Browsers, has no effect.

Usage:

$('.inputsClassName').customPlaceholder( params );

Params: { activeClass: 'className' }

The activeClass is the class for styling the placeholder text.

Requirements:

JQuery and Modernizr.
